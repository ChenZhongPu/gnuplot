# Greek Letters

You can use `{\Symbol <Alphabet>}` to display Greek letters. For example `{\Symbol a}` is for \\(\alpha\\).

## Upper case

| Alphabet | Full | Symbol |
|---------|---------| ----- |
| A | Alpha | A |
| B | Beta | B |
| C | Chi | X |
| D | Delta | \\(\Delta\\) |
| E | Epsilon | E |
| F | Phi | \\(\Phi\\) |
| G | Gamma | \\(\Gamma\\) |
| H | Eta | H |
| I | iota | I |
| K | Kappa | K |
| L | Lambda | \\(\Lambda\\) |
| M | Mu | M |
| N | Nu | N |
| O | Omicron | O |
| P | Pi | \\(\Pi\\) |
| Q | Theta | \\(\Theta\\) |
| R | Rho | P |
| T | Tau | T |
| U | Upsilon | \\(\Upsilon\\) |
| W | Omega | \\(\\Omega\\) |
| X | Xi | \\(\Xi\\) |
| Y | Phi | \\(\Phi\\) |
| Z | Zeta | Z |
